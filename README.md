## 悠悠鹽埕-鹽埕介紹網站

介紹鹽埕的景點、美食、歷史、交通方式，技術上主要使用Angular與RWD，網站請見：https://iamkyc.github.io/Yen-Chen-webpage/index.html

### 技術與嘗試

* 使用Angular達成呈列美食、搜索美食功能
* 響應式頁面(RWD)
* 使用google map api
* 以鮮明照片或顏色使網頁看起來較為活潑
* 在滾動滑鼠時顯示或消失navbar，並且使其動畫順暢
* 首頁icon的呼吸燈效果
* 回頂端按鈕的呈現與動畫
* 使用ajax從「公共運輸整合資訊流通服務平台」接收捷運到站時間


###語言
* html、css、javascript、jQuery
* Angular
* google map api
